Agentic AI in Software Engineering

Replication package for reviewers

This page documents the search, screening, and augmentation steps used to identify secondary studies on agentic AI in software engineering (SE).

Source
Scopus (metadata + full text)
Time window
Published after 2023
Final set
11 papers (Scopus + snowballing + SE2030)
Style inspired by the Agentic AI in SE living repository layout (single-page, card-based sections).

Goal

Provide transparent, step-by-step documentation of how we retrieved and selected secondary studies on agentic AI in SE, including intermediate lists and the final curated set.

Search protocol

Database

Scopus, searched in metadata + entire paper.

Search string

( "agentic AI" OR "AI agents" OR "autonomous agents" OR "software agents" )
AND ( "software engineering" )
AND ( taxonomy OR survey OR "secondary study" OR review OR "mapping study" OR "empirical study" )

The exact Scopus query as exported from the platform is available here: searchString.pdf.

Rationale

We adopted this string to capture terminological variability in how agentic AI is discussed in SE (including AI, autonomous, and software agents) while focusing on synthesis-oriented evidence (taxonomies, surveys, reviews, and mapping studies).

Filters

  • Publication year: after 2023
  • Venues: Computer Science and AI journals/conferences

After filtering: 233 papers (see CompleteList.csv).

Selection flow

1

Initial retrieval

Apply the Scopus query (metadata + full text) with year and venue filters. Output: 233 records in CompleteList.csv.

2

Title screening

First pruning by reading titles only. Output: 28 records in 28Selected.csv.

3

Eligibility check (full-paper)

Include: secondary studies on Agentic AI in SE. Exclude: papers not focused on SE activities/tasks. Output: 7 records in FinalList_scopus.csv.

4

Backward/forward snowballing

Add 2 relevant papers from references/citations. Output: FinalList_scopusPlusSnowballing.csv.

5

Targeted venue check (SE2030)

Complement with ACM TOSEM “2030 Roadmap for Software Engineering” special issue (Vol. 34, Iss. 5, 2025), adding 2 papers. Final output: 11 papers in FinalList_scopus Plus snowballing Plus SE2030.csv.

Reference venue page: ACM TOSEM, Volume 34, Issue 5 (2025).

How to reproduce

  1. Run the Scopus query with the exact string and filters reported above (after 2023; CS/AI venues; metadata + full text).
  2. Export results as CSV to obtain the initial list (expected size: 233, subject to Scopus indexing updates).
  3. Title-screen to obtain the 28-paper shortlist.
  4. Read full texts to apply eligibility criteria (include secondary studies focused on SE activities/tasks).
  5. Perform backward/forward snowballing on the eligible set.
  6. Check the TOSEM SE2030 special issue (Vol. 34, Iss. 5, 2025) and add relevant secondary studies.

Note: counts may vary slightly over time due to database updates; the CSVs in this package preserve the exact snapshot used.
