# Web materials

This folder contains structured material used by the Agentic AI in SE website.

- `taxonomy_categories.json`: final taxonomy categories and boundary cases.
- `extended_tetrad.csv`: expanded McLuhan Tetrad construction material.
- `research_agenda.csv`: open questions, testable questions, and suggested empirical methods.
- `validation_summary.csv`: expert elicitation summary and taxonomy-refinement note.
- `validation_note.md`: short explanation of the Passive Team-member pre-refinement boundary case.

The 11-paper scoping corpus from the first iteration is documented in the `data/` folder. The extended Tetrad and risk-benefit synthesis draw on broader literature surveyed during the second iteration as documented by the Tetrad traceability files.

- `risk_benefit_analysis.csv`: benefits, risks, and expected changes for each agent category, including not-applicable boundary cells.

- `tetrad_source_traceability.csv`: source-by-source traceability for the extended McLuhan Tetrad construction, including reference IDs, source type, and Tetrad dimension(s).
- `tetrad_source_summary.json`: machine-readable counts for the Tetrad source base.
